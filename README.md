# Drupal_Project
Project Overview
This project is a local Drupal 11 setup using DDEV and Docker for development purposes.
The project includes:
Drupal 11
Drush (Drupal CLI)
Admin Toolbar module
Gin Admin Theme
DDEV for local development environment
Requirements
Make sure the following are installed:
Docker
DDEV
Composer
WSL/Ubuntu (if using Windows)
Visual Studio Code (optional)
